🚀 Triangle Widget Features

Advanced capabilities of the Supply Chain Triangle visualization

📊 Real-time Updates

Triangle shape adjusts dynamically as metrics change, providing instant visual feedback on performance.

🎯 Target Overlays

Optional overlay showing target triangle shape for easy comparison with current performance.

📈 Historical Trends

Click any vertex to see historical trend data for that metric over time.

🔍 Drill-down Analysis

Click the triangle to access detailed breakdowns of each metric component.

🎨 Color Coding

Automatic color changes based on performance: green (good), amber (warning), red (critical).
